# 💎 Prism-QA: Professional Translation Auditor

[简体中文](./README.md) | [English](./README.md)

Prism-QA 是一款基于 DeepSeek-V3 引擎的工业级翻译质量审计工具。它支持多维度评分可视化，并能生成专业的 PDF 审计报告。
Prism-QA is an industrial-grade translation quality assurance tool powered by the DeepSeek-V3 engine, supporting multi-dimensional score visualization and professional PDF report generation.

---

## ✨ 特性 | Features

* **可视化频谱 (Visual Spectrum)**: 通过图表直观展示准确度、流利度、术语及风格得分。
* **扣分详解 (Deduction Analysis)**: 精确定位翻译错误，给出具体的扣分原因与位置。
* **PDF 报告 (PDF Export)**: 一键导出结构化的审计报告，方便离线查阅与分享。
* **安全设计 (Security)**: 采用环境变量管理 API Key，确保秘钥不泄露。
